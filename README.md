# Botman-Chatbot
This is a Chatbot named Botman. It educates users on cybersecurity. It teaches them about online threats and ways to stay safe online.

## Features
- Keyword-based responses
- Typing effect simulation
- Coloured console interface
- ASCII art UI
- Audio introduction
- Continuous conversation loop

## Technologies Used
- C#
- .NET
- GitHub Actions (CI)

## How to Run
1. Open project in Visual Studio
2. Build and run the application
3. Interact with BotMan via console
