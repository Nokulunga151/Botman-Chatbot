## Botman-Chatbot
This is a Chatbot named Botman. It educates users on cybersecurity. It teaches them about online threats and ways to stay safe online.

## Features
- Keyword-based responses
- Windows Form Application GUI
- Chat based layout
- Chat bubbles between user and chatbot
- ASCII art UI
- Audio introduction
- Continuous conversation loop
- Task creation for users to set reminders
- Quiz that will text users on the knowlege of cybersecurity

## Technologies Used
- C#
- .NET
- GitHub Actions (CI)

## Prerequisites
- Visula Studio 2026
- .NET 10.0
- Newtonsoft.Json Nuget package

## How to Run
- Open project in Visual Studio
- Build and run the application
- Interact with BotMan via GUI/WinForms
